Gradle

To install the library add: 
 
   ```gradle
   repositories { 
        jcenter()
        maven { url "https://jitpack.io" }
   }
   dependencies {
         implementation 'com.github.iamniklas:colorspaces:[VERSION]'
   }
   ```
